# Telecom Customer Churn Analysis
**End-to-End Data Portfolio Project**

### 🛠️ Tools Used
* **Excel:** Initial data auditing and cleaning.
* **MySQL:** Database management and churn metric verification.
* **Python (Pandas):** Exploratory Data Analysis (EDA) and correlation insights.
* **Power BI:** Interactive 6-page dashboard with AI-driven insights.
* **Canva:** Custom UI/UX design and iconography.

### 💡 Key Findings
* **High Risk:** Fiber Optic customers and those on Month-to-Month contracts.
* **Price Sensitivity:** Churn likelihood doubles when monthly charges exceed $68.
* **Support Gap:** Customers without Tech Support are 2x more likely to leave.
